# 智能无人系统（官方 未来-01）

> `[8+2+3 ▸ 未来 ▸ 01 / 03]`  ─ **L4 级无人公交示范** · **全国首个全空间无人体系示范场景** · 海陆空一体

## 定义

南沙 [[8-2-3-现代化产业体系]] **第一未来产业**，是国家《关于支持广州南沙放宽市场准入与加强监管体制改革的意见》(2024-01-09) 直接点名的"海陆空全空间无人体系"。覆盖无人车、无人机、无人船、eVTOL、管控平台、数字孪生、运营服务全栈。

## 核心原理

1. **海陆空全空间** ─ 不止地面 L4，水面无人船 + 低空无人机 + eVTOL 一体化场景
2. **场景反哺技术** ─ 全国首个"全空间无人体系示范场景" → 真实数据回流
3. **政策卡位** ─ 国家级《市场准入意见》直接授权先行先试

## 关键要素

| 要素 | 说明 |
|------|------|
| 无人车 | L4 级无人驾驶公交示范区 |
| 无人机 | 物流无人机、巡检无人机、应急无人机 |
| 无人船 | 港口、海洋测绘、海事执法 |
| eVTOL | 城市空中出行（与 [[低空经济与航空航天]] 重叠） |
| 平台 | 管控调度平台、数字孪生、运营服务 |
| 政策 | 《广州南沙海陆空全空间无人体系建设和低空经济高质量发展行动计划》2024-10 |

## 应用示例
