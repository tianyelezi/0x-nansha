# 细胞与基因 (CGT · 官方 未来-02)

> `[8+2+3 ▸ 未来 ▸ 02 / 03]`  ─ **40+ 家企业** · 研发-临床-应用全生态 · 多病种临床突破

## 定义

南沙 [[8-2-3-现代化产业体系]] **第二未来产业**，对应国家《推动未来产业创新发展实施意见》(2024) 中"生命健康"方向。已聚集**超过 40 家**细胞与基因治疗（CGT）企业，覆盖研发—临床—应用创新生态，**已实现多病种临床突破**。

## 核心原理

- **CGT 三段式** ─ 上游（载体 / 细胞工厂） + 中游（药物开发） + 下游（临床转化）
- **港澳药械通** ─ 借 [[生物医药产业]] 通道做 CGT 临床先行
- **公共平台** ─ CGT 公共服务、样本质控、合规评价

## 关键要素

| 要素 | 说明 |
|------|------|
| 细胞治疗 | CAR-T、TCR-T、NK、干细胞 |
| 基因治疗 | AAV 载体、基因编辑、mRNA |
| 临床转化 | 与 [[南沙生命科学城]]、6 家三甲、2 个国家医学中心联动 |
| 平台 | CGT 公共服务平台、样本库、质控、合规评价 |

## 应用示例

- **CAR-T 治疗罕见病** ─ 通过港澳药械通先行临床
- **基因编辑工厂化** ─ 与 [[合成生物产业]] 上游底盘细胞共平台
