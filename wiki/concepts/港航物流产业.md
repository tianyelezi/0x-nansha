# 港航物流产业 (PORT / 升级-1)

> `[8+2+3 ▸ 升级 ▸ 01 / 03]`  ─ 大湾区航运枢纽 ─ 智慧化、绿色化、国际化

## 定义

南沙最厚重的传统优势：**华南国际航运中心核心承载区**。8+2+3 升级方向 = **智慧港口 + 绿色港口 + 自由贸易港**。

## 核心原理

- **自动化码头** ─ [[南沙港]] 四期全球最大单体自动化码头
- **多式联运** ─ 海铁公水空五式联运
- **跨境贸易** ─ 自贸区 + CEPA + RCEP 红利

## 关键要素

| 要素 | 说明 |
|------|------|
| 码头 | 集装箱、汽车滚装、邮轮 |
| 多式联运 | 南沙港铁路、内河驳运 |
| 物流园 | 国际物流中心、保税仓 |
| 数字化 | 区块链单证、数字孪生 |
