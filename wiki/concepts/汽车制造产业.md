# 汽车制造产业 (AUTO / 升级-2)

*Automotive Manufacturing*


> `[8+2+3 ▸ 升级 ▸ 02 / 03]`  ─ 千万辆级广州车都的南极

## 定义

广州是中国汽车第一城。南沙是**广汽丰田 + 广汽埃安 + 智能网联**承载区，8+2+3 升级方向 = **新能源化、智能网联化、出口化**。

## 核心原理

- **整车 + 三电** ─ 与 [[新能源产业]] 电池闭环
- **车规芯片** ─ 与 [[半导体与集成电路产业]] 协同
- **智能驾驶** ─ 与 [[人工智能产业]] 模型耦合
- **滚装出口** ─ 与 [[港航物流产业]] 联动

## 关键要素

| 要素 | 说明 |
|------|------|
| 整车 | 广汽丰田、广汽埃安 |
| 三电 | 电池、电机、电控 |
| 智能 | 智能座舱、智驾域控 |
| 出口 | 南沙汽车滚装码头 |

## 应用示例

- **新能源车出海** ─ 通过 [[南沙港]] 滚装到中东、东南亚
- **SiC 800V** ─ 第三代半导体上车
