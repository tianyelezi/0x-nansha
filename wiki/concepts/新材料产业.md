# 新材料产业 (MAT / 战新-5)

*New Materials*


> `[8+2+3 ▸ 战新 ▸ 05 / 08]`  ─ 万物之基 ─ 高端化、绿色化、海洋化

## 定义

聚焦**电子化学品、先进高分子、复合材料、海洋工程材料、可降解材料**，作为 8 战新中最"硬底层"的基础产业，向 IC、新能源、航空航天、海洋装备多向供应。

## 核心原理

- **横向赋能** ─ 一种材料供 N 个下游
- **就地配套** ─ 缩短半导体、电池原材料链路
- **海洋特色** ─ 利用南沙海岸线发展海工材料

## 关键要素

| 要素 | 说明 |
|------|------|
| 电子化学品 | 光刻胶、抛光液 |
| 高分子 | 工程塑料、特种橡胶 |
| 复合材料 | 碳纤维、玻纤 |
| 海工材料 | 防腐涂层、深海钢 |
