# 船舶与海洋工程（官方 战新-02）

> `[8+2+3 ▸ 战新 ▸ 02 / 08]`  ─ **300 亿+ 集群** · "梦想"号 · 中船龙穴 · 广船 · 黄埔文冲

## 定义

南沙 [[8-2-3-现代化产业体系]] 第二战新航道。以**"梦想"号大洋钻探船**、**中船龙穴造船基地**、**广船国际**、**黄埔文冲**为代表的海工装备集群，已形成**300 亿元以上**产业规模。聚焦船型设计、总装建造、舾装配套、海工平台、海事服务、检验。

> 📌 与 v1.0 [[海洋经济产业]] 部分重叠 — 后者更宽（含邮轮、渔业），本页聚焦官方明示的"船舶 + 海工装备"核心。

## 核心原理

- **大国重器集聚** ─ "梦想"号 + 万米钻探船 + 大型 LNG 船
- **港工一体** ─ 龙穴造船基地与 [[南沙港]] 同岛布局
- **链向未来** ─ 海工是 [[深海深空]] 未来产业的硬件载体

## 关键要素

| 要素 | 说明 |
|------|------|
| 整船 | 大型集装箱船、LNG 船、邮轮、钻探船 |
| 海工平台 | 半潜式平台、FPSO、风电安装船 |
| 配套 | 舾装、船舶检验、海事服务 |
| 龙头 | 中船集团广船国际、黄埔文冲、龙穴基地 |

## 应用示例

- **"梦想"号大洋钻探船** ─ 国之重器，向 [[深海深空]] 延伸
- **海上风电安装船** ─ 服务珠江口与南海风电
- **超大型 LNG 船** ─ 国际市场份额
