# 绿色石化与新材料（官方 战新-06）

> `[8+2+3 ▸ 战新 ▸ 06 / 08]`  ─ **300 亿+ 集群** · 沙伯基础 · 巴斯夫 · 石化 → 新材料 → 终端

## 定义

南沙 [[8-2-3-现代化产业体系]] 第六战新航道，**300 亿元以上**集群。以沙伯基础（SABIC）、巴斯夫等国际化工巨头为支撑，形成**"石化原料 → 新材料 → 终端应用"**完整链条，覆盖汽车、电子、储能多个下游。

> 📌 v1.0 [[新材料产业]] 仅含"新材料"，本页按官方口径加上**绿色石化**上游。

## 核心原理

- **国际化工巨头集聚** ─ 沙伯基础、巴斯夫
- **横向赋能** ─ 一种材料对应多个下游：汽车、电子、储能、医疗
- **绿色化** ─ 低碳石化工艺、可降解材料

## 关键要素

| 要素 | 说明 |
|------|------|
| 石化原料 | 烯烃、芳烃、特种气体 |
| 高分子 | 工程塑料、特种橡胶、改性材料 |
| 功能材料 | 电子化学品（光刻胶、抛光液）、电池隔膜、复合材料 |
| 下游配套 | 汽车（[[智能网联与新能源汽车]]）、电池（[[新能源产业]]）、半导体（[[半导体与集成电路产业]]） |

## 应用示例

- **光刻胶** ─ 供 [[半导体与集成电路产业]]
- **电池隔膜 / 锂电材料** ─ 供 [[新能源产业]]
- **改性工程塑料** ─ 供 [[智能网联与新能源汽车]] 内饰与三电壳体
